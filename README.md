## Description

Fast, efficient, powerful, easy Need For Speed World core server

## Requirements

- nodejs >= 18.0
- mongodb >= 6.0

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
